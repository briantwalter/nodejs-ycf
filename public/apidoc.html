API Reference

This API is the simplest you could ever use. Create a HTTP GET request and get a JSON response.

Example HTTP GET request:

curl http://ycf.app.walternet.us/api

Example JSON response:

{
  "title": "Yoda's Cat Facts",
  "catfact": "Purring alwayss not indicate that a cat is happy.  Also purr loudly, cats will when they are distressed or in pain.  Hmmmmmm."
}
